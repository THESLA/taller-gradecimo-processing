# Taller Gradécimo Processing

Proyecto de taller de Processing para grado décimo.

## Contenido

- Scripts de generación musical con IA (MusicGen, Riffusion)
- Procesamiento de audio
- Material didáctico

## Requisitos

- Python 3.11+
- PyTorch 2.8.0+cu128
- Dependencias listadas en `requirements.txt`

## Herramientas

- **MusicGen (Meta)**: Generación de música con IA de alta calidad
- **Riffusion**: Generación experimental basada en espectrogramas
