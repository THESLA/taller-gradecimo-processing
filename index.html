Taller Gradécimo Processing

Generación musical con Inteligencia Artificial

Sobre el proyecto

Este repositorio contiene herramientas y scripts para la generación de música utilizando modelos de IA locales. Desarrollado como material didáctico para el taller de grado décimo, explorando la intersección entre creatividad musical y tecnología de aprendizaje automático.

Herramientas

MusicGen (Meta)

Modelo de generación musical de alta calidad. Produce audio realista a partir de descripciones textuales.

Riffusion

Generación experimental basada en espectrogramas y difusión. Ideal para explorar sonidos únicos.

Procesamiento

Herramientas de post-procesamiento de audio con libros, scipy y torchaudio.

Stack tecnológico

Python 3.11 PyTorch 2.8 + CUDA 12.8 NVIDIA RTX 5060 Transformers Diffusers Librosa

Cómo usar

1. Clona el repositorio:
git clone https://github.com/THESLA/taller-gradecimo-processing.git

2. Genera música con MusicGen:
python musicgen_generate.py "una canción alegre de cumbia con acordeón y tambora"

3. Explora y experimenta con diferentes prompts en español.
